### Delete duplicate rows

```*.sql
SELECT	<selected_column>,
COUNT	(ani_id) AS NumOccurrences
FROM	<table_name>
GROUP 	BY <selected_column>
HAVING 	(COUNT(<selected_column>) > 1 )


DELETE 
FROM	<table name> A
WHERE	ROWID > (SELECT min(rowid)
FROM	<table name> B
WHERE 	a.<selected_column> = b.<selected_column>);

```
